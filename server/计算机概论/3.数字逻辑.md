## 存储单元

1. R-S 锁存器（与非门实现）（S 输入对应的输出 a 为锁存器保存的值）

```
两输入两输出，其中输出是互补的。
R=0,S=0 没有意义，锁存器的状态不确定
R=0,S=1 置0
R=1,S=1 置1
R=1,S=1 保持
```

2. 门控 D 锁存器
